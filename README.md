# 韶乐隐私政策

**最后更新：2025年2月22日**

## 我们的承诺
韶乐是一款完全本地化的音乐播放器，我们不收集、不存储、不传输任何个人数据。

## 数据收集
我们不收集任何数据：无个人身份信息、无设备信息、无使用数据。

## 本地存储
应用仅在您的Mac本地存储：音乐库信息、用户设置、歌词缓存。

## 网络访问
仅在您主动使用时访问网络（在线歌词、AI识别、封面下载），不记录任何活动。

## 数据安全
所有数据仅存储在本地，不含追踪器或广告。
